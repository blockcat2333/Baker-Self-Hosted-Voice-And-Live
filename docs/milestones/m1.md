# Milestone 1

## Goal

Deliver a runnable engineering skeleton with clear boundaries and local infrastructure.

## Included

- monorepo structure
- root build, lint, test, typecheck scripts
- React web shell
- Electron shell reusing shared UI
- API / Gateway / Media placeholder services
- PostgreSQL + Redis compose setup
- shared protocol and SDK packages
- initial Drizzle schema
- minimal tests

## Explicitly Deferred

- auth implementation
- guild/channel CRUD
- text message persistence
- presence synchronization
- voice orchestration
- screen sharing
- retries, metrics, tracing

## Exit Criteria

- repo structure is coherent
- local setup path is documented
- services expose health endpoints
- gateway handshake uses shared protocol
- database schema package exists
- tests cover at least protocol parsing and media adapter boundary
