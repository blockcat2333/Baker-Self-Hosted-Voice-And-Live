# UI i18n (English + 简体中文)

## Scope

- UI text only (labels, headings, placeholders, buttons, helper copy).
- Do not translate: usernames, server names, channel names, user messages, logs, or raw error codes.

## Implementation

- Library: `i18next` + `react-i18next`.
- Locale detection:
  - first visit: detect from browser language
  - after manual switch: prefer stored selection in `localStorage` (`baker_language`) across refreshes
- Supported locales: `en`, `zh` (language-only; `zh-CN` resolves to `zh`).

## Key conventions

- Default namespace: `translation`.
- Key format: `feature.key_name` (lowercase, underscores).

## Terminology baseline

Use these translations consistently across the web client and the admin panel:

- Server: `Server` / `服务器`
- Channel: `Channel` / `频道`
- Voice: `Voice` / `语音`
- Livestream: `Livestream` / `直播`
- Screen share: `Screen share` / `屏幕共享`
- Control panel: `Control Panel` / `控制面板`
- Management password: `Management Password` / `管理密码`
- Sign out: `Sign out` / `退出登录`
- Save: `Save` / `保存`
- Cancel: `Cancel` / `取消`
- Retry: `Retry` / `重试`
